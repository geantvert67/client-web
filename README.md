### Installation

-   `npm install`
-   Créer un fichier `.env` à la racine du projet
-   Y ajouter le contenu suivant :

```
REACT_APP_API_URL=
```

### Usage

-   **Lancer en mode développement :** `npm start`
-   **Créer le dossier de production :** `npm run build`
